## 🧠 SAI — Flagship Project

> **Self-Adaptive Intelligence** — a modular cognitive framework that restructures how AI thinks, reasons, and evolves.

```
User Query
    │
    ▼
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   ANALYZE   │────▶│  PROMPTOR   │────▶│     SAI     │
│  clarifies  │     │ structures  │     │  executes + │
│   intent    │     │  the query  │     │   evolves   │
└─────────────┘     └─────────────┘     └─────────────┘
                                               │
                              ┌────────────────┼────────────────┐
                              ▼                ▼                ▼
                       Graph of Reason   Feedback Loop    Role Generation
```

**Core capabilities:**
- `.feedback_loop` — recursive self-improvement
- `.graph_of_reason` — multi-layer logical analysis
- `.auto_mpr` — metaphor-driven creative reasoning
- `.constraint_solver` — turning limits into growth points
